# Бюджетный помощник
## Уханова Екатерина, Кудашова Полина - поток ОП 2.1
Программа для отслеживания ежедневных расходов пользователя за месяц. Реализует префиксные суммы, линейный поиск, сортировку вставками, стек и дерево.

## Структура проекта:
- main.py - модуль для непосредственного взаимодействия пользователя с помощником, текстовый интерфейс
- budget_manager.py - модуль, объединяющий все остальные модули
- expenses.py - модуль, содержащий хранилище расходов и префиксные суммы
- categories.py - модуль, содержащий трекер категорий и сортировку вставками
- stack.py - модуль, содержащий стек для отмены операций
- tree.py - модуль, содержащий бинарное дерево расходов

## Функционал:
1. Добавить расход: вводится день, сумма, категория
2. Сумма за период: префиксные суммы 
3. Максимальный день: день с наибольшим суммарным расходом, используется линейный поиск
4. Категории: все категории по убыванию сумм, используется сортировка вставками
5. Отменить: отмена последнего добавленного расхода, используется стек
6. Все расходы: все записи в порядке возрастания дней, используется центрированный обход дерева
7. Расходы за день: все расходы конкретного дня, используется поиск в дереве
